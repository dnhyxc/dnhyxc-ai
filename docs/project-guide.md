# 产品功能详解与使用教程

本文面向普通用户，介绍本产品的主要功能、适用场景与操作方法，帮助你从“能用”到“用得顺手”。（内容会随版本持续更新）

---

## 1. 你可以用它做什么

你可以把它理解为一个“**带知识库的 AI 助手**”：

- **对话（Chat）**：进行日常问答、方案讨论、写作润色、代码与文档协作等。
- **知识库（Knowledge）**：用 Markdown 记录与整理资料（可云端/本地两种模式），并在写作与检索中复用。
- **英语学习**：按主题流式生成单词包与经典句；支持 JSON 导入与**资源库**分页浏览、拉取历史、Agent 多会话、收藏与导出 Word（DOCX）等（需登录后使用主能力）。
- **Markdown 增强**：支持数学公式、代码高亮、任务列表、Mermaid 图表等，适合写技术笔记与方案文档。
- **桌面端体验**：在桌面客户端可使用更多系统能力（例如全局快捷键、选择目录、开机自启等）。

---

## 2. 运行形态与差异（桌面端 vs 浏览器）

### 2.1 桌面端（推荐）

适合需要“更强本地能力”的用户：

- 可使用**全局快捷键**（跨应用生效）
- 可选择本地目录作为文件存储位置
- 可使用部分系统级能力（如开机自启等）
- 适合重度写作/长期知识管理

### 2.2 浏览器端

适合快速访问、临时使用：

- 功能以网页可实现的能力为主
- 一些系统级能力会提示“仅桌面端可用”
- **未登录时**：仍可浏览首页、打开**知识库（默认仅本地模式）**、阅读部分公开说明页（如产品指南、更新说明、服务政策等）；需要账号的主流程（如智能对话）会在进入后按策略引导登录。

## 3. 快速上手（5 分钟）

### 3.1 第一次使用建议

1. 先进入**对话**：尝试问一个你正在做的任务（例如“帮我写一份周报”、“帮我做一个方案对比”）。
2. 再进入**知识库**：创建一篇 Markdown 记录，把对话结论沉淀下来。
3. 如你经常写技术文档：试试 **任务列表**、**数学公式**、**Mermaid 图表**。

### 3.2 常见使用姿势

- **边聊边记**：先用对话梳理思路，再把结论写入知识库。
- **资料整理**：把外部资料（链接、摘要、要点）整理为 Markdown，形成可检索、可复用的笔记。
- **方案沉淀**：用 Markdown 写“背景—目标—方案—权衡—结论”，配 Mermaid 流程图/时序图。

### 3.3 首页「快速开始」：步骤卡与顶栏

首页「快速开始」区域里，**步骤列表**与**顶栏主按钮**分工不同，避免把「注册账号」和「直接进入对话」混在同一个入口上：

- **顶栏「快速开始」类主按钮**：仍用于**尽快进入智能对话**（进入 **`/chat`** 主界面），适合已经准备好直接开聊的用户。
- **步骤卡片（整行可点）**：例如「**注册账号**」一步会带你打开登录页并**直接呈现注册表单**（地址栏会带上 **`mode=register`** 查询参数，便于刷新或复制链接后仍停在注册视图）；另一步「**开始使用**」等与顶栏一致，进入对话主界面。
- **在登录页内切换「登录 / 注册」**时，界面与地址栏会同步更新；为减少历史记录堆叠，通常使用 **replace** 方式写入地址，行为以当前版本为准。

若你通过书签或外链打开 **`/login?mode=register`**，也会直接看到注册表单。

## 4. 对话（Chat）详细教程

### 4.1 基础对话

你可以直接输入问题或任务描述。为了获得更稳定的结果，推荐按以下结构提问：

- **背景**：我在做什么？现状是什么？
- **目标**：我希望得到什么输出？（结论/步骤/清单/文案/代码）
- **约束**：字数、风格、受众、是否需要可执行步骤、是否需要对比方案等
- **输出格式**：例如“用表格对比”、“按 1/2/3 步骤给我”

示例：

- 我在做一个产品需求评审。目标是输出一页 PRD 摘要，约束是 300 字以内，格式用要点列表。

### 4.2 流式输出与停止/续写

- **流式输出**：回答会边生成边展示，方便快速阅读。
- **停止生成**：如果你已经看到想要的方向，可以停止，避免继续跑偏或过长。
- **续写**：当你希望在当前回答基础上继续展开，可以续写。
- **分支与重生成**：同一会话内可在消息树形态下分支或重生成；分享只读页与在线阅读在复杂分支场景下也会尽量保持消息顺序与排版一致（与下文「分享会话」说明一致）。

服务端对话生成已统一经硅基流动兼容接口调度，你侧仍按上述方式使用即可，无需额外配置。

### 4.3 联网检索与引用

当你需要：

- 获取最新信息（例如某个产品/框架的近期变化）
- 需要来源佐证（例如写报告、写评审材料）

你可以明确要求“请联网检索并给出引用”。你也可以要求：

- 只引用权威来源
- 给出可点击的引用
- 先列引用再总结

### 4.4 附件与 OCR（图片/截图文字）

当你有截图、图片中的文字内容需要分析时，可以上传附件并要求：

- “提取文字并总结要点”
- “识别表格并转换为 Markdown 表格”

上传成功后，附件会出现在输入框上方；**图片类附件可点击卡片打开预览**。若文件名含中文，预览与下载会自动处理编码；Web 与桌面端均可在对话内直接查看，无需手动复制图片链接到新标签页。在浏览器访问线上站点时，预览走与对话相同的站点地址，一般无需单独配置图片静态路径；部署新版本后若仍无法预览，请确认前后端均已更新并重启服务。

### 4.5 桌面端语音输入（Tauri，桌面壳）

当你使用**桌面客户端**进行对话时，底部输入区可在**文本输入**与**语音输入**之间切换：

- 将指针移到**圆形主按钮**上，会展开「输入模式」菜单，在其中选择文本或语音。
- 选择**语音**后，点击主按钮开始说话，识别出的文字会**持续写入**输入框；再次点击主按钮结束录音，当前框内文字即为本次会话中已识别的内容（过程中可随时编辑再发送）。
- 将应用界面语言切换为英文后，上述菜单与相关提示也会随语言切换显示。

## 5. 知识库（Knowledge）详细教程

知识库用于“长期沉淀”。你可以把它当成你的个人文档空间。

### 5.1 两种模式：云端 vs 本地

- **云端模式**：适合多设备同步与集中管理。
- **本地模式**：适合把知识库放在你自己的文件夹里（Markdown 文件可被任何编辑器打开）。

如果你未登录，知识库会默认走**本地模式**，并隐藏不适用的入口（例如回收站）。

### 5.2 创建与编辑

一般流程：

1. 新建一篇文档（标题 + 正文）
2. 用 Markdown 写内容
3. 保存（手动保存 / 自动保存）

建议的文档结构模板：

- 标题
- 背景
- 目标
- 结论（先写结论）
- 过程与论据
- 待办（任务列表）
- 参考链接

### 5.3 保存、覆盖保存与自动保存

你会遇到三种保存体验：

- **手动保存**：适合你希望“确认一次再落盘/提交”的场景。
- **覆盖保存**：你希望同名时直接覆盖（适合持续维护同一篇文档）。
- **自动保存（防抖）**：当你停止编辑一段时间后自动保存，减少忘记保存的风险；它会尽量避免打断你输入。
- **保存前自动格式化**：无论是手动保存还是自动保存，落盘前都会先对当前正文做一次与编辑器内「格式化」一致的排版整理，再写入存储。

建议：

- 写长文时开启自动保存
- 做结构化整理时用手动保存更可控

### 5.4 本地文件夹：扫描、打开、删除与外部编辑器

当你选择本地文件夹作为知识库来源时：

- 列表会递归扫描该目录下的 Markdown 文件
- 你可以打开文件进行编辑与保存
- 也可以在外部编辑器中打开（适合你习惯用 Cursor 等工具编辑）
- **删除更可控**：在桌面端（Tauri，桌面壳）删除时，会根据条目的来源提供不同选项：
  - **本地文件夹列表中的文件**：删除仅影响本地磁盘文件（不会删除云端数据）。
  - **云端知识条目且已关联到本地同名文件**：删除确认框会提供 **「删除本地文件 / 删除在线文件 / 同时删除」** 三种操作，方便你按需处理（例如只清理云端记录但保留本地笔记）。

### 5.5 回收站（如可用）

当你使用云端管理时，删除内容可能会进入回收站，便于误删恢复。

### 5.6 文档内 AI 助手（需登录）

当你**已登录**并在知识库中编辑文档时，可在编辑器底部使用**知识库文档助手**：结合当前 Markdown 正文与 AI 多轮对话，便于润色、总结与答疑。

- **未登录不展示**：未登录时不会出现该助手区域，避免占用版面或产生无效入口；本地编辑知识文档不受影响。
- **长对话滚动**：助手对话列表变长后，可在输入框右上方使用**滚动到底部 / 滚动到顶部**按钮，快速跳转；流式生成时若你上滑阅读历史，可再滚回底部或点「置底」以继续跟随最新输出。
- **选区发送至助手**：在 Markdown 编辑器中选中一段正文后，可将选中文本**发送到**底部文档助手，再结合 **AI 模式** 或 **RAG 模式** 提问，减少来回复制粘贴；若多次框选重叠内容，产品侧会对重复上下文做收敛，具体以当前版本为准。
- **多轮上下文**：AI 模式下，助手会结合当前文档已保存的会话历史继续回答（在系统 token 预算内自动裁剪较早轮次）；RAG 模式每轮主要依据当次检索到的知识库片段作答。服务端生成链路已与主站对话统一为硅基流动兼容接口，交互方式不变。
- **桌面端文本/语音与界面语言**：在**桌面客户端**且已登录时，助手底部输入区与对话页类似，支持通过悬停主按钮展开「输入模式」菜单，在**文本输入**与**语音输入**之间切换；菜单与相关提示会随你在设置中选择的**界面语言**（如中文/英文）显示。语音模式下，识别文字会**实时填入**输入框，停止录音后保留当前内容以便你修改或发送。
- **「生成目录」自动写入文首**：在 **AI 模式**（非 RAG）下，可使用快捷卡片 **「生成目录」**；模型流式结束后，系统会把抽取出的目录块插入左侧编辑器**文首**，并在目录列表上方加上 **`## 目录`** 二级标题。若文首已是 **`## 目录`**，会提示不再重复插入；若仅有锚点列表或用了其它级别的「目录」标题，会**补全或规范为** `## 目录` 而不重复整段列表。
- **流式回复**：助手生成过程中仅展示正文与「正在生成中…」状态，不展示模型内部思考链；加载指示可正常动画。

## 6. Markdown 写作能力（让文档更强）

### 6.1 任务列表（TODO）

适合写计划、排期与验收点，例如：

- [ ] 需求确认
- [ ] 原型评审
- [x] 技术方案评审

### 6.2 数学公式（KaTeX，数学公式渲染）

适合写算法、推导与公式说明。你可以用行内或块级公式表达。

### 6.3 代码块与高亮

适合写技术文档、记录命令、粘贴片段。不同语言会有不同高亮风格。

### 6.4 Mermaid 图表（流程图/时序图）

适合把复杂流程“画出来”，例如：

- 流程图：业务流程、发布流程
- 时序图：前后端交互、接口调用链
- 状态图：状态流转

建议：

- 先用对话让 AI 帮你生成 Mermaid 草稿
- 再把草稿粘到知识库里微调

## 7. 快捷键（提高效率）

### 7.1 全局快捷键（桌面端）

在桌面端可设置全局快捷键，用来执行常用动作（例如隐藏/显示、刷新等）。  
当你设置的快捷键与已有快捷键冲突时，系统会阻止保存并提示冲突项，避免出现“一组快捷键触发多个动作”的问题。

### 7.2 页面内快捷键（知识库）

知识库页提供页面内快捷键（例如保存、清空草稿等），只在知识库页面内生效，避免在其他页面误触。

## 8. 设置项建议（推荐配置）

### 8.1 文件存储

- 桌面端可选择更适合你的默认存储目录（例如同步盘/工作目录）

### 8.2 启动与关闭行为（桌面端）

- 若你希望随时可用：开启开机自启、关闭时最小化到托盘
- 若你更注重资源释放：关闭时直接退出程序

### 8.3 大模型（自定义配置）

1. 登录后进入 **设置**，在顶栏选择 **大模型**（路径 **`/setting/llm`**）。
2. 填写 **API Key**（若本地开发环境已配置默认密钥，首次打开可能已自动填入，保存后以你的账号配置为准）、**Base URL**、**模型名称**。
3. **Base URL** 与 **模型名称** 可直接输入，也可点输入框右侧按钮展开预设：可选「智谱 GLM」「硅基流动」或「DeepSeek」等配对地址与模型；从一侧选择预设时另一侧会自动同步，仍可手填其它服务商地址。**切换预设或模型时会清空 API Key**，需重新填写对应服务商的密钥。
4. 填写完整后点击 **保存** 即启用自定义模型。底部若显示绿色「**某某** 模型配置生效中」，表示**当前登录账号**在对话、知识库助手、英语学习等模块会使用该配置；其它账号互不影响。
5. 未保存自定义时：底部灰色提示「当前默认 **某某** 模型生效中」——**有效会员**对应硅基流动默认模型，**非会员**对应智谱 GLM；与 **§2** 会员权益衔接。
6. 有未保存修改时，底部会提示「有未保存更改，请保存」或「可保存设置让自定义模型生效」；任一必填为空时提示补全三项并无法保存。
7. 需要改回服务器默认时：点击 **恢复默认配置**（会关闭自定义并恢复与会员状态对应的默认表单项）。

说明：API Key 输入框支持右侧眼睛图标切换显示/隐藏。

**向量模型（知识库 RAG，与上方大模型独立保存）**

1. 在同一页面向下滚动至 **向量模型** 区块。
2. 填写 **API Key**、**Embedding 接口地址**（完整 URL，非 Base URL 拼接）、**Rerank 接口地址**、**向量模型名称**、**重排模型名称**、**向量库名称**；各字段支持预设下拉（如硅基流动、bge / Qwen3、默认库名等）。**切换预设或模型时会清空向量 API Key**，需重新填写。
3. 点击 **保存向量配置** 即启用；底部绿色提示表示当前账号的知识库向量化与 RAG 检索走你的配置。未保存时灰色提示使用服务端默认。
4. 每次保存会将当前「向量库 + 模型」**累积记录**；下方 **已纳入检索的向量库** 列表展示历史库。RAG 会并行检索列表中的库，并**始终包含**系统默认 bge 库（即使用户未单独保存过）。
5. 新保存的文章写入**当前选定的向量库**；旧库中的文章仍可通过多库 RAG 被检索到。
6. 需要恢复时点击 **恢复向量默认**（不影响上方对话大模型配置）。

### 8.4 语音设置

1. 进入 **设置**，在顶栏选择 **语音设置**（路径 **`/setting/cloud-tts`**）。**所有用户**均可打开该页。
2. 页面上方为 **本机语音设置**：按女声/男声分组选择浏览器本机朗读音色，可试听；偏好按当前登录账号保存在本设备。
3. **有效会员**在本机区可打开 **使用本机语音朗读**，在下方 **云端语音设置** 可打开 **使用云端语音朗读**；两个开关**互斥**，决定英语学习里点喇叭时默认走本机还是云端（云端暂时不可用时可能回退本机）。**非会员**无此开关，仅本机朗读。
4. 开启 **使用云端语音朗读** 后，可调整模型、音色、语速、音量、音高、情感、音频格式、语言增强等；**采样率、码率、声道** 等高级项一般保持默认。未开启云端开关时，下方参数区不可编辑。
5. 点 **朗读参数** 标题旁的 **?** 可查看云端各字段说明。
6. 云端区可 **试听**（走云端合成）或 **恢复默认参数**；自定义参数**保存在你的账号下**（换设备登录同一账号可同步）。
7. **英语学习中的播放**：**非会员**使用 **本机语音设置** 中的浏览器音色。**有效会员**由上述两个开关决定本机或云端；本机区 **试听** 始终走本机音色。

### 8.5 账户头像（云存储）

1. 登录后进入 **账户** 或 **个人资料** 相关页面（侧栏头像区域亦可进入账户设置）。
2. 在头像区域选择本地图片上传；上传过程需保持已登录状态。
3. 上传成功后会显示预览，点击保存后头像在侧栏与资料页同步更新。
4. 若图片无法显示：请确认已重新上传（旧云存储链接可能失效）；Web 站点请使用与对话相同的站点地址访问，避免在浏览器中单独打开已过期的外链。
5. 在头像大图预览中可点击工具栏**下载**保存到本地；上传区悬停时的下载按钮行为相同。完成后会有成功或失败提示；若下载失败，请确认仍已登录且云图可正常预览。

说明：智能对话中的**聊天附件**与头像一样保存在腾讯云对象存储；历史会话里若仍为旧式本地路径，展示与解析会尽量兼容，新上传一律走云存储。

### 8.6 会员充值（Stripe）

1. 登录后进入 **`/pay` 会员充值**（也可从 **`/profile` 个人主页** 点击「购买会员」）。
2. 在三种固定套餐中选择其一：**月度 ¥9.9（30 天）**、**季度 ¥25.9（90 天）**、**年度 ¥99.9（365 天）**；价格由产品定价，页面**不可编辑金额**。
3. 点击「在页面内打开收银台」，按 Stripe 嵌入式表单完成支付（需服务端与前端配置 Stripe 密钥）。
4. 支付成功后会**自动跳转到个人主页**：可看到**金色「会员」徽章**与**金色「会员有效期至 …」**文案；若已是会员再购，会在**原到期日之后**续期叠加。
5. 会员到期后，再次进入个人主页或重新登录会恢复为非会员展示，并可再次购买。

## 9. 常见问题（FAQ）

### 9.1 为什么有些功能提示“仅桌面端可用”？

浏览器受安全限制无法直接调用某些系统能力（例如全局快捷键、文件系统目录选择等）。桌面端拥有这些能力，因此体验更完整。

### 9.2 我写的 Markdown 为什么显示和预期不一样？

建议：

- 检查代码块围栏是否闭合
- Mermaid/公式是否符合语法
- 尝试将内容拆成更小段落排查
- 如果你在知识库编辑器里粘贴了大段 ` ```tsx`，并且围栏内容包含 “```mermaid” 等字面量，格式化时可能会出现围栏反引号数量变化；属复杂围栏边界情况，建议先备份正文或分段粘贴再格式化。

### 9.3 如何把对话结论沉淀成可复用的知识？

推荐流程：

1. 对话中让 AI 输出“结论 + 关键要点 + 待办清单”
2. 复制到知识库文档中
3. 补上你的背景、约束与最终决策

### 9.4 换账号登录后还会看到上一个账号的内容吗？

不会（在**同一浏览器标签页、未整页刷新**的前提下）。当你**登出**、用**另一个账号登录**，或登录态因 **401** 失效被清理时，应用会自动清空与上一账号相关的前端展示缓存，例如知识库未保存草稿、文档助手与 RAG 对话、英语学习 Agent 聊天、进行中的单词包/经典句进度等。若你**仅更新同一账号的资料或会员状态**（仍是同一个用户），这些编辑与对话**不会被**清空。若仍有异常，可尝试刷新页面或重新登录。

## 10. 术语小字典

- **Tauri（桌面壳）**：把 Web 前端打包成桌面应用的一种方式。
- **SSE（Server-Sent Events，服务端事件）**：服务器持续向客户端推送数据的方式，常用于流式输出。
- **OCR（Optical Character Recognition，光学字符识别）**：从图片中识别并提取文字。
- **GFM（GitHub Flavored Markdown，GitHub 风格 Markdown）**：Markdown 的常用扩展规范。
- **Mermaid（图表语法）**：用文本写流程图/时序图等的语法。
- **Debounce（防抖）**：在停止操作一段时间后才执行动作，常用于减少频繁保存/请求。
- **RAG（Retrieval-Augmented Generation，检索增强生成）**：在回答前先从你的资料中检索相关片段，再结合模型生成答案的一种方式（本产品在知识库助手中提供相关模式）。

## 11. 分享、RAG 模式与界面语言

### 11.1 分享会话（只读浏览）

当你希望把一段对话给同事或客户**只读查看**时，可使用产品内的**分享**能力生成链接；对方在浏览器中打开即可阅读，无需安装桌面客户端。分享页的消息顺序会与你在对话里看到的顺序对齐（含分支/重生成等复杂情况）；若原对话里用户消息带有图片或文件附件，分享页也会显示相同的附件卡片，可预览或下载。页面排版、用户消息中的代码块与知识正文预览等与在线会话阅读体验一致。

在知识库中已保存到云端的文章，可通过 **分享** 生成 **`?type=knowledge`** 的公开链接；打开后标题下方会显示 **「更新 …」**，时间与你在 **云端知识库列表** 中看到的最近更新时间一致（按你设备所在时区的本地时间展示）。请先 **保存到知识库** 再分享，以确保对方看到的是已落库正文与时间。

### 11.2 知识库助手：AI 模式与 RAG 模式

在知识库中编辑并已登录时，底部**文档助手**除默认的 **AI 模式**（结合当前 Markdown 正文多轮对话）外，还可切换到 **RAG 模式**（检索增强）：系统会结合检索到的资料片段作答，适合「对着资料问具体问题」的场景。模式切换、输入区与流式输出与对话页风格一致，检索命中内容通常会以引用或摘录形式参与回答，便于核对出处。

### 11.3 界面语言（中文 / 英文）

在**设置**中将界面语言切换为**英文**或**中文**后，对话页、知识库助手、菜单与常见提示会随语言包切换显示。个别能力还依赖运行环境（例如桌面端语音需要系统麦克风权限）。若你发现某处文案未随语言切换，多半是后续新增界面尚未接入翻译键，可向维护者反馈。

## 12. 编辑器里还能做什么（知识库 Markdown）

除正文编辑、分屏预览与「修改对照（Diff）」外，你还可以：

- **右键上下文菜单**：在编辑区选中文本后，可通过快捷菜单完成常用编辑动作；与底部操作栏能力互补。
- **Mermaid 图表缩放与预览**：复杂图表可放大、预览查看，减少误读。
- **代码块格式化**：在支持的围栏语言下可对代码块做格式化；若粘贴了带嵌套围栏的复杂片段（例如大段 ` ```tsx`），可能出现反引号数量变化等边界情况，建议先备份或分段处理。

## 13. 英语学习（单词包、经典句与收藏）

在侧栏进入**英语学习**专区（需登录后主流程可用）后，你可以围绕一个**主题**生成结构化学习内容，并管理个人收藏。

### 13.1 单词包与经典句（流式生成）

- **按主题生成**：为单词包或经典句等能力填写主题、数量等条件后发起生成；生成过程以**流式**方式在界面中逐步展示，便于中途阅读。
- **取消与错误提示**：生成过程中可按产品提供的操作取消请求；若网络或服务异常，界面会给出可读错误说明，具体文案以当前版本为准。
- **主检索与按需联网、RAG**：生成主阶段由助手汇总要点；**联网检索**会在模型判断需要时触发（例如主题涉及时效、日期类信息），减少无意义的例行检索。若你已在知识库中沉淀资料，也可结合 **RAG（检索增强生成）** 等模式，让回答带可核对的引用与摘录（与知识库助手域能力一致）。

### 13.2 快捷意图（工具栏「芯片」）

工具栏提供若干**快捷意图**（形如可点的「芯片」条目），为本次发送内容附加固定前缀或意图说明；再次点击同一项通常可**取消选中**。文案会随你在设置里选择的**界面语言**切换。

### 13.3 左栏表单与返回页面

在英语学习路由内填写的主题、数量等左栏条件，在你**暂时离开再返回**该页面时，一般会尽量恢复，减少重复填写（与生成流式状态的全局单例等实现协同，以实际版本为准）。

### 13.4 收藏、抽屉与导出 Word（DOCX）

- **收藏**：单词或经典句支持一键收藏；收藏列表在**抽屉**中分页浏览与管理（折叠状态等交互以实现为准）。
- **批量取消收藏**：在收藏抽屉中可多选条目，经**二次确认**后批量取消收藏；单条也可快速移除。
- **导出 DOCX**：可将**单词收藏**或**经典句收藏**分别**一键导出为 Word（DOCX）** 文件。导出由服务端按你的账号汇总至多约 **3000** 条（按收藏时间倒序），与抽屉里分页浏览的「当前页」无关；在**浏览器**与**桌面客户端（Tauri）** 下均走二进制下载与本地保存流程，桌面端若已有同名去重或提示，以实现为准。单词收藏导出可包含**词性（pos）**等字段（以当前版本为准）。

### 13.5 资源库与 JSON 导入

除「按主题拉取」外，你还可以把本地 **JSON 单词包 / 经典语句包** 导入为长期保存的**资源库**：

- **导入入口**：在英语学习左栏的「单词库 / 语句库」区域进入导入；独立页路径为 **`/english-learning/import`**，通过查询参数 **`kind=vocab`**（单词）或 **`kind=classic`**（经典句）区分类型。
- **导入流程**：支持拖拽或选择 `.json` 文件；可在编辑器中预览与手改 JSON；校验通过并填写标题后保存。上传较大文件时产品会走适合大体积的上传方式，避免普通请求体大小限制导致失败。
- **导入后**：保存成功后会跳转到**资源库**并选中刚创建的库，便于立刻浏览词条。
- **文件名预填**：选择文件后，导入标题可自动用**去掉后缀的文件名**预填，减少手输。

### 13.6 资源库浏览与管理

- **分页列表**：选中左侧某个单词库或语句库后，右侧词条以**分页**加载，可滚动「加载更多」，适合大包体资料。
- **会话内记忆**：在同一会话（不整页刷新）内切换库或暂时离开再返回时，会尽量**恢复已加载页数与滚动位置**，减少重复翻页。
- **删除词库**：可对不再需要的单词库执行删除，经**二次确认**后移除库及其词条（经典语句库逻辑对称，以实现为准）。
- **收藏星标**：资源库与包列表中的词条会按需查询是否已收藏；星标会随列表增量加载逐步点亮，避免一次查询过多词条。

### 13.7 拉取历史、结果页与停止生成

- **历史抽屉**：单词包 / 经典句拉取区可打开**历史记录**列表，查看以往生成会话；进行中的会话会显示进行中标识，且通常**不可删除**，直至结束。
- **打开历史**：点击某条历史会进入**流式结果页**查看当时生成的词条，**不会**用历史主题覆盖你左栏正在填写的新拉取表单。
- **删除历史**：可对已结束的历史记录删除；删除后相关已落库词条一并清理（以实现为准）。
- **结果页**：页头展示本次主题与联网摘要等信息；进行中的拉取与纯历史查看在分页与交互上有所区分，避免切换历史后丢失当前进度。
- **停止生成**：流式拉取过程中可停止；停止后界面以已生成内容为准，且一般不会因「用户主动停止」再弹出刺眼的错误提示。

### 13.8 英语学习 Agent（多会话）

- 在英语学习主界面可使用 **Agent 对话**：支持**多路会话**，可在历史抽屉中分页浏览并切换；点「新对话」会清空当前展示，**首条消息发送时**再创建服务端会话。
- **快捷意图**仅影响当轮模型输入，**不会**写入会话历史正文。
- 将 Agent 回答保存到知识库后，产品可引导你进入**知识库**继续编辑（以实现为准）。

### 13.9 列表与左栏交互细节

- **列表折叠**：已拉取的单词或经典句卡片列表标题旁可**收起 / 展开**网格，默认展开；新一轮拉取开始时会自动展开，便于看到新内容。
- **快捷意图折叠**：左栏快捷意图较多时，默认只展示前几条，可展开查看全部，与主列表折叠交互风格一致。
- **词性展示**：单词包词条可显示英文缩写形式的**词性（如 n、v、adj）**，便于区分同一词形的不同用法。

### 13.10 网络不稳定时的体验（尤其桌面端）

在**桌面客户端**访问远程服务时，列表首屏、加载更多或收藏状态查询若遇到瞬时网络失败，产品会尽量**自动重试**只读请求，并以**可读文案**提示失败原因，避免把底层技术错误原文直接展示给你。收藏、取消收藏等写入操作通常不会重复提交，以免产生副作用。

### 13.11 听写与拼写练习（单词与经典句）

- **单词入口**：在**单词收藏**底部、**单词资源库**（左侧词库列表悬停耳机图标，以及**右侧当前词库词条列表顶栏**）、**单词词包结果页**顶栏、**单词历史拉取记录抽屉**，以及**错题集页底栏**（单词 Tab），可进入听写或「看中写」练习。
- **经典句入口**：在**经典句收藏**底部、**语句资源库**（左侧语料库悬停耳机图标与右侧语句列表顶栏）、**经典句词包结果页**顶栏、**经典句历史抽屉**，以及**错题集页底栏**（语句 Tab），可进入听写或「看中写」练习（按整句英文作答）。
- **练习设置**：进入练习后先选模式、题量与顺序；页顶会显示**词表来源名称**，右侧显示词表总量（单词为「128 词」，经典句为「20 句」）。
- **练习流程**：逐题作答；听写模式可播放读音后输入英文，拼写模式根据中文释义输入英文。**判分**忽略大小写；单词另忽略句末标点，经典句另忽略全部标点（仍须字母与空格一致）。
- **提示**：作答过程中可点题卡右上角 **提示** 展开线索，再次点击 **收起提示**。**单词**：听写可查看中文释义与音标（仍不显示英文词），拼写可在释义下方查看音标。**经典句**：听写可查看中文释义、出处或赏析（仍不显示英文原句），拼写可在释义下查看出处或赏析；本题无可用线索时按钮不可用；进入下一题后提示会自动关闭。
- **答错后重答**：**第一次**答错时只标红你的答案并自动展开提示（仍不显示英文正确答案），以字段表展示线索并可播放；底栏左侧播放、中间引导文案、右侧圆形 **看答案**（与播放钮同款样式）。点 **看答案** 或按 **→** 进入完整揭示时，**若正在朗读会继续播完**（两页共用播放状态）。**再次答错**则直接显示完整正确答案（含英文、音标/分词/例句或语句出处等）。可点 **再试一次** 在本题重答，**上一题**（非首题）回到上一道并重练，或 **下一题** 记入本轮并继续。**键盘**：作答 **Enter** 检查；**Shift + 空格** 播放/停止；首次答错与完整揭示后 **↑** 上一题、**←** 再试、**↓** 下一题；首次答错另可用 **→** 看答案。题卡顶栏会标明**单词听写**、**语句听写**、**单词看中写**或**语句看中写**。
- **看中写提示与播放**：顶栏点 **提示** 展开后，除音标等线索外还可**播放**读音；首次答错提示区同样可播放。
- **听写播放**：进入题目、换题与听写 **再试一次** 时自动连续播放 **3 次**（间隔约 **3 秒**），可点停止。仅**未展开提示时**题卡中央**大号播放钮**亦为 **3 次**；**Shift + 空格** 与答错页手动播放为 **1 次**（未展开提示时 Shift+空格 亦为三连播）。**看答案** 不会打断进行中的朗读。
- **快捷键说明**：练习页顶栏右侧 **?** 图标可打开下拉列表，按「作答中 / 首次答错 / 看答案后」查看 **Enter** 与方向键对应操作（图标展示，无路径说明）。
- **练习报告（结算）**：本轮结束后进入报告页，顶部展示正确率与分项统计；下方**作答明细**先列出错误词、再列出正确词（正确项以**绿色左边框**区分，错误项为**红色左边框**），可点击播放读音复习。
- **结算后操作**：可**重练错题**、**继续练习**（在词表范围内尽量抽取尚未练过的词）、**重新设置**；可将本轮错词**加入错题集**；可**进入错题集**查看已保存的错词；**返回**为页眉左侧**返回图标**。
- **返回去向**：从**英语学习首页**的历史记录进入练习时，返回会回到**首页**；从**词包结果页**的历史抽屉进入时，若练习的是另一条记录，返回后仍保持原先选中的那条记录；从**单词或语句错题集**进入练习后，返回会回到对应的错题集页。

### 13.12 错题集（单词与语句）

- **进入方式**：在**英语学习首页**侧栏点击**单词错题集**或**语句错题集**（直达对应分类）；也可打开错题集页后在顶栏 Tab 切换（布局与「我的收藏」一致）。同一侧栏还提供 **今日复习**（见 **§13.14**）。
- **顶栏**：标题旁显示**总数 / 已加载**；右侧切换**单词错题集** / **语句错题集**。
- **单词内容**：错词含词面、音标、释义、例句与上次错拼；**语句内容**：错句含英文、中文释义、出处、赏析与上次错拼；均支持播放、勾选**全选已加载**、**移除所选**。
- **再次练习**：底栏右侧 **听写 / 拼写** 以当前 Tab 的错题集为词表来源；无错题时禁用。
- **与结算页关系**：练习报告可先**加入错题集**再**进入错题集**核对；已存在的词形或同一句若错拼与上次不同会**更新上次错拼**，错拼相同则不变。

### 13.13 语句错题字段说明

- **语句错题列表**展示英文原句、中文释义、出处、赏析与上次错拼；播放为整句朗读。其余操作见 §13.12。

### 13.14 今日复习（间隔复习）

- **位置**：**英语学习首页**左栏 **今日复习** 区块（玫瑰金色图标），分别显示 **单词复习**、**语句复习** 按钮及各自待复习数量；无到期项时对应按钮不可用。
- **来源**：通常来自你在听写/拼写结算页**加入错题集**的条目；仅**新加入**或**错拼有变化**的错题会进入「今日待复习」。很久以前已在错题集、但尚未产生复习计划的条目，一般不会自动出现在今日队列。
- **开始练习**：点击 **单词复习** 或 **语句复习** 后进入听写/拼写**设置页**（与收藏、错题集入口一致），请自行选择 **听写 / 看中写**、**题量**（如 10～50），再点 **开始练习**。复习队列由系统按到期顺序抽取，设置页不展示「顺序 / 随机」选项。
- **结算**：完成一轮后，系统根据每题对错调整下次复习时间；答对的条目会从「今日待复习」中移出，侧栏计数会刷新。复习场次结算页**不会**再次提供「加入错题集」（条目本身来自错题集）。
- **返回**：从侧栏进入的练习，结束后 **返回** 会回到 **英语学习首页**。
- **继续练习**：结算页 **继续练习** 会在尚未练过的到期项中尽量再抽一批；若今日已无更多到期项，会提示无更多可复习内容。

### 13.15 今日记词

- **位置**：**英语学习首页**左栏 **今日记词** 区块；可设置 **每轮词数**（10～50），点 **开始记词** 进入 **`/english-learning/daily`**。
- **流程**：介绍页 → 认读（听发音、看释义）→ 点 **考考我** 做四选一 → 正/错反馈 → **下一词** 直至完成；练完后写入记词记录并纳入间隔复习，可在 **记词记录** 页查看。
- **四选一**：错误选项优先来自本轮其它词的释义，并尽量与当前词 **词性、释义长度** 相近，减少同一轮内重复出现相同干扰项。
- **底栏按钮**：介绍页 **开始记词** 与听写/拼写设置页 **开始练习** 底栏间距一致；认读与反馈页的 **考考我 / 下一词 / 完成** 为全宽主按钮，无额外深色边框间隙。

### 13.16 资源库编辑与公共词库

- **编辑入口**：在**单词库 / 语句库**左侧列表中，悬停或选中某个库卡片时，删除按钮左侧有**编辑**图标。**创建者**对**尚未公开**的自有库可见；**已公开**的资源库仅**超级管理员**可见编辑（用于维护公开状态）。
- **修改名称**：点击编辑后，在弹窗中可修改**资源库名称**（最多 **50** 字，输入框右侧显示当前字数）；仅**创建该库的用户**可保存改名；浏览他人公共库时名称输入框为只读。有未保存变更时，在弹窗内按 **Enter** 可快捷保存。
- **设为公共**（超级管理员）：若你的账号为**超级管理员**，同一弹窗内会出现 **设为公共资源库** 开关；开启并保存后，全站登录用户都能在资源库中看到并使用该库（浏览词条、听写/拼写、今日记词等）；关闭则恢复为仅创建者可见。**新导入的库默认不公开**，须由超管在此手动开启。
- **删除与标签**：你只能删除**自己创建**的库；他人公共库无删除按钮。列表上的 **公共** 小标签在库已公开时对**所有登录用户**可见，便于识别全站可读资源库。

### 13.17 首页左栏布局与交互

- **卡片分区**：英语学习首页左侧各功能（今日记词、快捷意图、单词库/语句库、按主题拉取、收藏、今日复习、错题集等）以独立**卡片**呈现，标题左侧有彩色图标区，整体风格与右侧 Agent 区、知识库侧栏的细边框浅底一致；各区块按钮仍保留原有配色（如今日记词、词包拉取等色相不同）。
- **导入 JSON 示例**：在单词库/语句库卡片内，**JSON 格式示例默认收起**；点击「展开导入示例」类标签可查看示例并再次点击收起，便于对照格式又不占满首屏。
- **快捷意图排布**：快捷意图默认展示前几条，点 **展开** 后芯片在窄侧栏以**双列**排布；侧栏拉宽时会自动增加列数，避免意图条目挤成单列竖排。
